# Payroll Manager Role Center

The **Payroll Manager** Role Center is the home page for payroll users in OnePayroll. It provides a dashboard with key payroll information, quick-access cues, and navigation to all payroll functions.

## Headlines

The top section displays contextual headlines that highlight items requiring attention:

| Headline | What it shows |
|----------|---------------|
| **Next Payroll Dates** | Upcoming payroll runs and their scheduled dates |
| **Open Payroll Runs Without Payments** | Payroll runs that are open but have not yet had payments generated |
| **New Hires** | Employees with upcoming employment dates |
| **Terminations** | Employees with upcoming termination dates |
| **Upcoming Birthdays** | Employee birthdays coming up |
| **Upcoming Anniversaries** | Employee work anniversaries coming up |

Select a headline to drill down to the relevant employee list or payroll runs page.

## Activity cues

The activity section shows counts of key items organized in two groups:

### Employees

| Cue | Shows |
|-----|-------|
| **Onboarding** | Employees in onboarding status (employment date in the future) |
| **Active** | Currently active employees |
| **Offboarding** | Employees with a future termination date |
| **Inactive** | Employees with a past termination date |

### Payroll

| Cue | Shows |
|-----|-------|
| **Journals** | Open manual payment journal entries |
| **Open** | Payroll runs in Open status |
| **Approved** | Payroll runs in Approved status |
| **Posted** | Payroll runs in Posted status |

Select any cue to open the corresponding list page filtered to that status.

## Navigation

The Role Center provides quick access to all OnePayroll areas:

### Embedded lists

- **Employees** — full employee list
- **Pay Types** — pay type definitions

### Sections

| Section | Pages |
|---------|-------|
| **Employees** | Employee List, Employee Types, Departments, Work Locations, Job Titles |
| **Journals** | Manual Payment Journal |
| **Benefits** | Benefit Types, Benefit Contracts |
| **Payroll** | Payroll Runs, Pay Groups, Pay Cycles, Pay Units, Payroll Setup |
| **Garnishments** | Garnishment Types |

### Create new

- **New Employee** — opens a new employee card
- **New Payroll** — opens the new payroll report to start a payroll run

### Reports

The following reports are available directly from the Role Center:

- **Payroll Register** — detailed register of payroll entries by employee
- **Payroll Register Difference** — compares two payroll runs showing differences
- **Cost of Labor** — labor cost analysis by employee, department, and pay type
- **Benefit Summary** — benefit analysis with contributions and limits
- **Pay Type List** — cross-employee comparison of pay type amounts

### History

- **Payroll Runs** — all payroll runs
- **Payroll History** — historical payroll entries
- **Payment Entries** — payment distribution records
- **Snapshots** — employee data snapshots

## Additional parts

The Role Center also includes standard Business Central components:

- **My Tasks** — user task list
- **Report Inbox** — scheduled report results
- **Power BI** — embedded Power BI report part (if configured)
